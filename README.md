# AI创作信息日报

面向影视导演、编导、叙事创作者、AIGC 内容团队的每日情报简报。

## 栏目结构
1. **行业与平台动向** — 大模型、平台能力、融资、政策、生态变化
2. **工具与能力更新** — 对创作链路真正有帮助的能力更新
3. **方法与经验** — 可执行、可迁移、可复用的创作方法与工作链路
4. **作品 / 案例** — 值得拆解的作品、传播样本

## 时间窗口
每天 08:00 ~ 次日 08:00（Asia/Shanghai），只收录 24h 内发生的信息。

```

  YYYY-MM-DD.md    # 每日日报
```
